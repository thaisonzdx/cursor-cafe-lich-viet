# Lich Viet

Vietnamese lunar calendar app. View lunar dates, Can Chi, traditional holidays, and manage personal events.

**[lichviet.thaison.me](https://lichviet.thaison.me)**

## Features

- Monthly calendar with solar and lunar dates
- Can Chi (Heavenly Stems & Earthly Branches) for year, month, and day
- Vietnamese lunar holidays highlighted
- Local event management (stored in browser)
- 4 themes: Light, Dark, Lotus, Jade

## Stack

Next.js, React, TypeScript, Tailwind CSS, shadcn/ui, [@dqcai/vn-lunar](https://www.npmjs.com/package/@dqcai/vn-lunar)

## Development

```bash
npm install
npm run dev
```
